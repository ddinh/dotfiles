# dotfiles

My dotfiles configured using [Ansible](https://docs.ansible.com/ansible/latest/index.html).

# install

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ddinh/dotfiles/master/bootstrap.sh)"
```

# configuration
- zsh
- ssh
- git 
- brew

# requirements

- Ubuntu 20.04 LTS (WSL)
